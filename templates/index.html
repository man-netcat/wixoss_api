Search a card by id by using /cards

Accepted GET parameters are:

Name GET Parameter
Card Id id
Card Name name
Card Type type
LRIG Type/Class class
Card Text text
Color color
Level level
Cost cost
Grow Cost grow_cost
Team team
Power power
Limits limits
Timing timing
Power Text power_text
Flavor Text flavor_text
Product product
Rarity rarity
Artist artist

For disjunctive search, use &or

You can get card images by using /img/<cardid>
